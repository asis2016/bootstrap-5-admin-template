Dependencies & Plugins

The adminAM depends on the following frameworks and plugins:

  • Bootstrap v5.3.2
  • jQuery v3.6.4
  • Chart.js v4.4.1
  • jVectorMap v2.0.5
  • plotly.js v1.58.5
  • Bootstrap Icons v1.11.2
  • Google Fonts
